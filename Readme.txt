# StudySync Website

## Description
This is a basic HTML webpage for "StudySync," designed to showcase a seamless learning platform. 
The page includes sections such as the header, navigation bar, main content, features, testimonials, and a footer.
It integrates CSS for styling and uses images to enhance the visual experience.

## Features

1. Navigation Bar: Links to Home, Features, Pricing, Blog, About, and Contact pages.
2. Hero Section: A welcoming message with links to start or take a tour.
3. Trusted Brands Section: Logos of trusted brands like Google, Microsoft, etc.
4. Feature Cards: Highlight the advantages of the platform, such as personalized learning.
5. Testimonials: Customer feedback section.
6. Newsletter: A form for users to subscribe to the latest updates.
7. Footer: Includes company information, product links, and social media icons.
## File Structure

- `index.html`: Main HTML file for the website.
- `study sync.css`: External CSS file for styling.
- Image files: Various images (logo, icons) used across the webpage.

## How to Run
1. Clone or download this repository.
2. Ensure all image and CSS files are in the correct directories.
3. Open `index.html` in any web browser to view the website.

## Technologies 
- HTML5
- CSS3
- External images for logos and icons
